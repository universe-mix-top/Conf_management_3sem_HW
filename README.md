## Доступные пораметры запуска
-read_path_file [локальный путь к файлу] - путь к файлу из которого читается код<br/>
-write_path_file [локальный путь к файлу] - путь к файлу в который будет записон вывод програмы

## Типы данных
var [имя] [содержание] - константы (нельзя изменить)<br/>
[имя] = [содержание] - переменная (может быть изменено)

## Операции
@([+/-/*/mod/min] [содержание]) - выпоняет соотвествующую операцию, в содержани может быть имя константы/переменной или число разделенные пробелом<br/>
{[содержание]. [содержание]. ...} - массив с неограниченным количеством констант/переменных, чисел или операций

## Значения
[1-9][0-9]* - формат чисел<br/>
[a-z]+ - формат имен констант/переменных

## Коментарии
После символа ' правая часть строки до конца считается коментарием

## Пример кода
var con 5;<br/><br/>

fon = @(+ con 4);<br/>
gen = @(* @(+ con 5) 4); 'коментарий <br/>
gen = @(min @(+ gen 5) 14 104);<br/><br/>

list = {1. @(min 3 2 4). {@(+ 1 2 3). 6}}<br/>
